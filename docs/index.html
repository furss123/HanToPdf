HanToPdf Project

HanToPdf

한글 파일(.hwp, .hwpx)을 PDF로 변환하는 Windows 데스크톱 앱

주요 기능

  • 드래그앤드롭으로 한글 파일 추가
  • 개별 PDF / 하나의 PDF로 합치기
  • 파일 이름 정렬 (오름차순·내림차순)
  • PDF 화질 설정 (원본화질 · 고화질 · 중간 · 저화질)

온라인에서 작업하기

이 저장소는 GitHub와 연동되어 있습니다. 브라우저에서 코드를 수정하거나 Codespaces로 개발 환경을 열 수 있습니다.

GitHub 저장소 Codespaces에서 열기 웹에서 파일 편집

Windows 앱 실행

PDF 변환은 한컴오피스(한글) COM 자동화를 사용하므로 Windows PC에서 exe로 실행해야 합니다.

로컬 빌드: powershell -File build.ps1

⚠️ 브라우저 페이지에서는 HWP→PDF 변환이 불가합니다. 변환 기능은 반드시 Windows 데스크톱 앱을 사용해 주세요.
